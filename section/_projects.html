Grep42

A freelance work search engine that is updated every 15 mins. Currently shows jobs sources from Elance and oDesk. I came up with the concept, then designed and developed it.


FlickD!

FlickD! is a movie collection manager for large movie collections. It can add new movie names from the media directly to the collection. FlickD! will help you find the movie you wish to see from the collection and even tell you the disk it's on.


LibSmsSharp

LibSmsSharp? is a .NET library written in C# which allows you to implement SMS functionality in your program. In its current state, it supports sending and receiving SMS and has a very flexible architecture allowing the feature set to be extended based on the phone-family and the AT commands supported by it.
